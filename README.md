# SR OS custom gRPC services

Introduced in SR OS 26.3.R1, custom gRPC services may now be created, 
backed by Python applications, to implement any gRPC protobuf package.

The official documentation can be find on the [Nokia documentation portal](https://documentation.nokia.com/sr/26-3/7x50-shared/system-management/grpc.html#custom_grpc_packages_services_and_rpcs).

This repository provides some short examples of custom gRPC services 
that can be run on an SR OS device.
